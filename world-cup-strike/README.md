# World Cup Mini Game

一个世界杯主题的弹射撞球小游戏 Demo：先从 2026 世界杯 48 队中选择我方球队，进入大赛模式查看完整赛程，然后通过双方各 3 名球员的物理弹射比赛逐步冲击冠军。

## 运行

### 网页版

直接打开：`/Users/lianghui/Documents/足球/world-cup-mini-game/web/index.html`

网页画布采用 iPhone 竖屏分辨率 `390 x 844`，会按浏览器窗口等比例缩放。

### 微信小游戏版

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本文件夹：`/Users/lianghui/Documents/足球/world-cup-mini-game`。
4. AppID 可使用测试号或游客模式。需要真机预览时，再把 `project.config.json` 里的 `appid` 改成你有开发者权限的小游戏 AppID。

## 操作

- 我方回合点击并拖拽任意存活球员，松手后沿虚线方向弹射。
- 球员撞到敌方球员会按攻击力和撞击速度造成伤害。
- 足球先进入对方球门的一方获胜；一方全员生命耗尽也会判负。
- 大赛模式中，结束后点击屏幕会回到赛程页继续下一阶段。

## 流程

- 选择球队：在 48 队轮播中选择我方球队。
- 大赛模式：按 2026 世界杯赛制模拟 12 个小组、32 强、16 强、8 强、半决赛和决赛。
- 赛程结果：可切换查看小组赛、32进16、16进8、8进4、半决赛、决赛的对阵、比分和完成进度。
- 玩家比赛：遇到我方球队的赛程时点击“开赛”，进入 VS 对撞展示，再进行三对三弹射撞球。

## 主要文件

- `game.js`：微信小游戏入口。
- `js/main.js`：Canvas 绘制、触控输入、球员 AI、比分和比赛状态。
- `game.json`：竖屏小游戏配置。
- `project.config.json`：微信开发者工具项目配置。
