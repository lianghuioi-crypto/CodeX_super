# 绯衣鬼戏剧情体验器

基于飞书剧情脚本文档整理的手游剧情体验器原型。当前版本参考小游戏“机智逃脱”的剧情玩法，使用 Canvas 绘制剧情场景、角色立绘、对话和基础交互。

## 当前内容

- 开场戏台演出与“绯衣娘子”传闻
- 倚云楼门口身份交涉
- 戏台尸体三处勘验：喉间、衣襟、右手
- 疑点收集完成后的案情判断
- 两段二合合成流程：长柄竹镊、黑色粉末
- 第一关结束点：前往后台追查铜镜鬼影
- 版本更新日志弹窗和界面内更新日志按钮
- 当前说话人高亮、脚下光晕、头顶对话泡泡
- 对话框角色头像 icon

## 运行方式

微信小游戏：

1. 用微信开发者工具打开本目录。
2. 编译运行，入口为 `game.js` -> `js/main.js`。

浏览器预览：

```bash
python3 -m http.server 8087
```

然后打开 `http://localhost:8087/`。

## 目录说明

```text
├── game.js                         微信小游戏入口
├── index.html                      浏览器预览页
├── web-game.js                     浏览器预览入口
├── js
│   ├── main.js                     微信小游戏剧情入口
│   └── story
│       ├── story-data.js           第一关剧情、交互点、合成链数据
│       └── story-engine.js         Canvas 剧情体验器引擎
├── images/characters               角色透明立绘与头像 icon
└── audio                           旧示例音频，当前剧情原型暂未依赖
```

## 后续替换资源建议

- 先在 `story-data.js` 中稳定剧情节点、交互点和合成链。
- 再把 `story-engine.js` 中的场景色块绘制替换为正式背景、角色立绘、物件图。
- 若新增关卡，优先追加新的 `scenes` 数据，保持引擎逻辑不变。
